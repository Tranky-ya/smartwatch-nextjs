# 📡 COMANDOS DE UBICACIÓN PARA RELOJES 4P TOUCH

## Comandos Comunes:

1. **DW** - Location Now (actual)
   Formato: `[3G*IMEI*0002*DW]`
   
2. **CR** - Center Number (ubicación + respuesta)
   Formato: `[3G*IMEI*0002*CR]`
   
3. **CENTER** - Ubicación central
   Formato: `[3G*IMEI*0006*CENTER]`
   
4. **POSITION** - Solicitar posición
   Formato: `[3G*IMEI*0008*POSITION]`

5. **UD** - Upload Data (forzar envío)
   Formato: `[3G*IMEI*0002*UD]`

6. **LK** - Link/Location request
   Formato: `[3G*IMEI*0002*LK]`

## Probar comandos:

Ver en logs qué comando funciona mejor con tus relojes específicos.
